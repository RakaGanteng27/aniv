# Kartu Anniversary — aniv.html

Cara pakai:

- Buka file `aniv.html` di browser (double-click atau buka lewat server lokal).
- Untuk mengganti nama atau tanggal, edit bagian `<div class="names">` dan `<div class="date">` dalam file.
- Klik kartu untuk membuka/tutup. Gunakan tombol "Cetak / Simpan" untuk menyimpan sebagai PDF atau mencetak.
- Tombol "Bagikan" menggunakan Web Share API (tersedia di beberapa browser mobile).

Catatan:
- File ini adalah halaman statis; tidak perlu backend.
- Jika ingin mengunduh sebagai gambar, gunakan ekstensi atau tool pihak ketiga (mis. html2canvas) — saya bisa menambahkan contoh bila diinginkan.
